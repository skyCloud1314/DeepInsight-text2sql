# PDF导出视觉重叠修复规格

## 介绍

PDF导出功能中AI思考过程与模型理解方式部分存在视觉重叠问题，需要彻底解决布局和间距问题，确保各个部分有清晰的视觉分离。

## 术语表

- **AI思考过程**: 显示模型推理过程的蓝色背景框
- **理解方式**: 包括"其他可能的理解方式"和"选中的理解方式"两个部分
- **视觉重叠**: 不同内容区域在PDF中出现重叠或间距不足的问题
- **ReportLab**: Python PDF生成库
- **ParagraphStyle**: ReportLab中定义段落样式的类

## 需求

### 需求1: AI思考过程独立显示

**用户故事**: 作为用户，我希望AI思考过程在PDF中有独立的显示区域，与其他内容有明显的视觉分离。

#### 验收标准

1. WHEN PDF包含AI思考过程时，THE系统SHALL在思考过程前后添加足够的空白间距
2. WHEN AI思考过程显示时，THE系统SHALL使用独特的背景色和边框样式
3. WHEN思考过程结束时，THE系统SHALL确保与下一个内容块之间有至少40像素的间距
4. THE思考过程样式SHALL包含上边距、下边距和内边距设置
5. THE思考过程标题SHALL显示用户配置的模型名称

### 需求2: 理解方式部分清晰分离

**用户故事**: 作为用户，我希望"其他可能的理解方式"和"选中的理解方式"在PDF中有清晰的分离和独立的样式。

#### 验收标准

1. WHEN显示其他理解方式时，THE系统SHALL在该部分前添加至少50像素的间距
2. WHEN显示选中理解方式时，THE系统SHALL使用不同的颜色主题（绿色）
3. WHEN理解方式部分开始时，THE系统SHALL添加明显的视觉分隔符
4. THE其他理解方式SHALL使用灰色主题样式
5. THE选中理解方式SHALL使用绿色主题样式

### 需求3: 商业洞察部分独立布局

**用户故事**: 作为用户，我希望商业洞察部分与AI思考过程和理解方式部分有明确的分离。

#### 验收标准

1. WHEN显示商业洞察时，THE系统SHALL确保与上一个内容块有至少35像素间距
2. WHEN商业洞察结束时，THE系统SHALL在理解方式部分前添加足够间距
3. THE商业洞察标题SHALL使用标准的标题样式
4. THE商业洞察内容SHALL使用正常的段落样式

### 需求4: 页面布局优化

**用户故事**: 作为用户，我希望PDF的整体布局合理，避免内容过于紧密或重叠。

#### 验收标准

1. THE系统SHALL为每个主要内容块定义明确的边距规则
2. THE系统SHALL确保相邻内容块之间的最小间距不少于20像素
3. WHEN内容块包含背景色时，THE系统SHALL增加额外的间距
4. THE系统SHALL在每个对话结束后添加分隔线和适当间距
5. THE页面边距SHALL设置为合理的值（上下0.5英寸）

### 需求5: 样式一致性

**用户故事**: 作为用户，我希望PDF中的样式在中文字体和默认字体版本中保持一致。

#### 验收标准

1. THE中文字体版本和默认字体版本SHALL使用相同的间距设置
2. THE所有样式定义SHALL包含完整的间距属性（spaceAfter, spaceBefore）
3. THE背景色和边框样式SHALL在两个版本中保持一致
4. THE字体大小和缩进SHALL在两个版本中保持一致
5. THE颜色主题SHALL在两个版本中保持一致

### 需求6: 测试验证

**用户故事**: 作为开发者，我希望有完整的测试来验证PDF布局的正确性。

#### 验收标准

1. THE系统SHALL提供PDF导出测试用例
2. THE测试SHALL验证各个内容块的间距设置
3. THE测试SHALL检查样式定义的完整性
4. THE测试SHALL验证不同内容组合下的布局效果
5. THE测试SHALL包含视觉重叠检测机制
